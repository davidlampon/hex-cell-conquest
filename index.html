Hex Cell Conquest

Territorial battle with predator-prey dynamics

Controls

  • Space - Pause/Resume
  • R - Reset game
  • 1 / 2 / 3 - Speed (0.5x / 1x / 2x)

Predator-Prey Chain

🟠 Orange beats Gray
⚪ Gray beats Cyan
🔵 Cyan beats Orange
Hunters get 40% speed boost!

Agent Modes

  • ◎ Hunt - Seeking enemy territory
  • ⚔ Chase - Pursuing prey (can kill!)
  • ⚠ Flee - Running from predator
  • ⚔⚔ Gang Up - Two vs weakest

Special Mechanics

  • Kill Streaks: Chain kills for bonuses!
  • Last Stand: <15% territory = Berserker mode (2x power)
  • Power Scaling: Leaders get stronger, weak get penalties
  • Victory: Eliminate enemies below 5% territory

Power-Ups (Spawn every ~15s)

  • 🛡 Shield: 5s invulnerability
  • ⚡ Speed: +50% speed for 8s
  • 💣 Nuke: Instant 10% territory conversion
  • ✨ Resurrection: Instantly respawn dead ally

Territory Movement Bonuses

  • Own Territory: +20% speed (home advantage)
  • Enemy Territory: -10% speed (deep strike penalty)

Evolution System

  • Recruit (0-2 kills): Normal
  • Veteran (3-5 kills): +10% size
  • Elite (6-9 kills): +20% size, ⭐ icon
  • Legend (10+ kills): +30% size, 👑 icon
